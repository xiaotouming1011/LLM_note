####  Skill 和 MCP 之间是什么关系？

**Skill 是能力 / 任务编排，MCP 是外部能力的标准接入通道；Skill 在上层 “做什么 / 怎么做好”，MCP 在下层 “怎么连 / 怎么调用”，两者是分层互补、常组合使用的关系。**

- **Skill（应用层/编排层）**：Agent 内部的**能力单元 / 任务模板**，定义 “**会做什么、步骤流程、规则规范**”（偏业务逻辑、工作流、领域知识）
- **MCP（通信层，协议层）**：**跨进程 / 跨服务的标准通信协议**，定义 “**如何安全、统一地调用外部工具 / 资源**”（文件、数据库、API、第三方服务）

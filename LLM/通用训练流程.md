# 数据Pipeline

**正确的训练数据 Pipeline 顺序：**

1. **数据集**加载
2. **数据读取与预处理**
   1. 加载路径
   2. 区分训练集，测试集
   3. 标签映射：正面负面/value,rejection/评分
   4. 清洗文本、去符号、转小写		
3. **分词器**
   1. 把句子切成单词列表
4. **词表构建 / 词嵌入模型**
   1. 给每个单词分配索引
   2. 给词汇表中的词匹配预训练向量
5. **数据加载器（DataLoader）**
   1. 自动分批次
   2. 打乱数据
   3. 不同长度补齐，转换为张量，搬到GPU

# 模型Training Loop

1.创建/初始化模型

2.开启训练模式

3.清空梯度

4.前向传播

5.算损失

6.反向传播（算梯度）

7.梯度裁剪

8.更新模型参数

9.累计损失

10.创建优化器optimizer

​	o.模型参数（权重与偏置）

​	a.学习率

​	b.损失函数

​	c.优化器选择（SGD/Adam/AdamW/RMS）

​	d.学习率调度器：学习率如何衰减
